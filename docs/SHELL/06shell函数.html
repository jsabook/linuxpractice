06shell函数

函数的定义

自定义函数

function fname(){
shell command1
shell command2
}

函数的执行

fname

第二种定义方式

fname(){
shell command1
shell command2
}

直接将function省掉

函数的参数

如果想要给函数增加参数

函数作用范围的变量

local 变量名

函数的参数

$1 $2 $3 ... $n
